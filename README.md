# nosbin

The decentralized pasting platform, built on nostr

### Features

-   [x] Markdown Support
-   [x] Use anonymous pubkey
-   [x] Use own pubkey
    -   [x] bech32 support
    -   [x] nox2x support
-   [x] Posting to multiple Relays
-   [X] Preview before posting
-   [ ] Comments
-   [ ] Reactions
-   [ ] Profiles
-   [ ] Search
-   [ ] Zaps
-   [ ] LN Pastes (Atomic Multipath Payments)
